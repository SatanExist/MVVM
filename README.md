  ##  MVVM: Система Заказа Напитков
  
Этот репозиторий содержит C++ и Blueprint логику для системы заказа напитков/аддонов, реализованную по паттерну Model-View-ViewModel (MVVM).

* **Unreal Engine Version:** 5.5 
* **IDE:** Visual Studio 2022 (с установленной рабочей нагрузкой "Game development with C++").

* Проект требует активации следующих официальных плагинов:
    1) UMG Viewmodel


  ##  Пошаговый Запуск

Поскольку это C++ проект, он не может быть запущен напрямую. Необходимо сгенерировать файлы решения.

1.  **Клонирование Репозитория:**
    ```bash
    git clone [ваша ссылка на HTTPS/SSH]
    ```
2.  **Генерация Файлов Visual Studio (.sln):**
    * В папке проекта найдите файл `[ВашПроект].uproject`.
    * **Правой кнопкой мыши** -> **Generate Visual Studio project files**.
3.  **Сборка C++:**
    * Откройте сгенерированный файл `.sln` в Visual Studio.
    * Установите конфигурацию: **`Development Editor`** | **`Win64`**.
    * Соберите решение: **`Build`** -> **`Build Solution`**.
4.  **Запуск:** После успешной сборки откройте проект двойным кликом по файлу `.uproject`.
